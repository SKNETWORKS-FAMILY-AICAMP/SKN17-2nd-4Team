# 2. 프로젝트 개요

### ▶️ 프로젝트 명 
# **떠날까? 말까?**
SNS 사용자 데이터를 기반으로 페이스북 이탈 가능성 예측 모델 제공
<br>
<br>

### ▶️ 프로젝트 소개 
해당 프로젝트는 SNS 사용자 데이터를 기반으로 개인 특성을 분석하여 향후 플랫폼 이탈 가능성을 예측하는 것을 목표로 함. 특히, 페이스북을 1순위로 선택한 사용자를 기반으로 페이스북 사용자 이탈 가능성을 예측함. 
<br>
<br>

### ▶️ 프로젝트 필요성(배경)
